## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Technologies Used

- React/Next.js for frontend
- Node.js for backend
- JavaScript/TypeScript
- AI integrations (ChatGPT, Claude APIs)
